    - onPositionChange?: (position: { paragraphIndex: number; charOffset: number }, cfi: string) => void
    - onTocLoaded?: (toc: NavItem[]) => void

    - onRenditionReady?: (rendition: Rendition) => void

    Implementation:
    - useRef for containerRef (HTMLDivElement), bookRef (Book), renditionRef (Rendition)
    - useState for isLoaded (boolean)
    - useEffect [url]: create ePub(url), store in bookRef
    - On book.ready: extract book.navigation.toc, call onTocLoaded(toc)
    - Call book.renderTo(containerRef.current, { width: '100%', height: '100%', flow: 'paginated' })
    - Register themes: light (bg:#ffffff, text:#1a1a1a), dark (bg:#1a1a1a, text:#e8e8e8), sepia (bg:#f4ecd8, text:#5b4636)
    - Call rendition.themes.select(theme)
    - Wire rendition.on('relocated', cb): extract location.start.cfi, location.start.percentage, call onProgressChange and onPositionChange
    - Call rendition.display() — after it resolves, setIsLoaded(true)
    - If initialPosition exists after display(), call onRenditionReady(rendition) so parent can handle restore
    - Cleanup: return () => { renditionRef.current?.destroy(); bookRef.current?.destroy(); }
    - useEffect [theme]: renditionRef.current?.themes.select(theme)

    Export a useImperativeHandle ref method navigateTo(href: string) that calls rendition.display(href).

    Render: <div ref={containerRef} className="h-full w-full" /> with skeleton overlay when !isLoaded
  </action>
