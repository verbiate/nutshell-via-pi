## Layout Contract

### Focal Point

**Unchanged from Phase 2:** Book content (EPUB iframe) fills the entire viewport. All new panels and toolbars are subordinate overlays that do not compete for attention.

### New Overlay: Floating Toolbar

| Property | Value |
|----------|-------|
| Position | `fixed` (parent document, not inside iframe) |
| Z-index | `70` (above chrome z-50, below Sheet z-100) |
| Height | `h-9` (36px) |
| Padding | `px-2 py-1` |
| Border radius | `rounded-md` |
| Background | `bg-popover` with `shadow-lg border border-border` |
| Arrow/pointer | None (clean rectangle, centered above selection) |
| Animation | `animate-in fade-in zoom-in-95 duration-150` |
| Exit animation | `animate-out fade-out zoom-out-95 duration-100` |

**Positioning algorithm (from RESEARCH.md):**
1. Compute iframe-local selection rect via `Range.getBoundingClientRect()`
2. Add iframe offset to get viewport coordinates
3. `top = iframeTop + rect.top - toolbarHeight - 8px`
4. `left = iframeLeft + rect.left + rect.width / 2 - toolbarWidth / 2`
5. **Flip:** If `top < toolbarHeight + 16`, place below: `top = iframeTop + rect.bottom + 8px`
6. **Clamp:** `left = max(8px, min(left, viewportWidth - toolbarWidth - 8px))`

**Content layout (left to right):**
```
[Highlighter icon] "Highlight"   |   [Sparkles icon] "Explain"
```
- Divider: `w-px h-4 bg-border mx-1`
- Each action is a `Button variant="ghost" size="sm"` with icon + label

**SearchSnippet pseudocode:**
```tsx
function SearchSnippet({ text, match }: { text: string; match: string }) {
  const parts = text.split(new RegExp(`(${escapeRegex(match)})`, 'gi'));
  return parts.map((part, i) =>
    part.toLowerCase() === match.toLowerCase() ? (
      <span key={i} className="search-match">{part}</span>
    ) : (
      <span key={i}>{part}</span>
    )
  );
}
```

## Interaction Patterns

### Text Selection Flow

| Step | Action | Result |
|------|--------|--------|
| 1 | User drags to select text in EPUB iframe | `rendition.on("selected")` fires |
| 2 | Handler captures CFI range, selection rect | FloatingToolbar becomes `visible=true` |
| 3 | Position computed from rect + iframe offset | Toolbar appears above (or below) selection |
| 4 | User clicks "Highlight" | POST highlight, `rendition.annotations.highlight()`, toolbar hides |
| 5 | User clicks "Explain this to me" | POST to `/api/explainers/generate`, ExplainerPanel opens (Current tab), SSE streams |
| 6 | User clicks elsewhere or presses Escape | Toolbar hides, selection clears |

**Selection guard:** Ignore selections shorter than 3 characters. If `rect.width < 2 || rect.height < 2`, do not show toolbar (Safari collapsed selection guard).

**Toolbar debounce:** No debounce on appear (immediate). Hide is immediate on click-outside or Escape.

### Search Flow

| Step | Action | Result |
|------|--------|--------|
| 1 | User clicks search icon or presses `Cmd/Ctrl+K` | SearchPanel opens, input auto-focused |
| 2 | User types query | Debounced 300ms, min 3 chars |
| 3 | Search runs over cached TXT | Results appear in ScrollArea |
| 4 | User clicks result row | `viewerRef.navigateTo(cfi)`, panel closes |
| 5 | User presses Escape | Panel closes |

**Keyboard shortcut:** `Cmd/Ctrl+K` opens search panel. `Cmd/Ctrl+F` also opens search panel (alias).

### Bookmark Flow

| Step | Action | Result |
|------|--------|--------|
| 1 | User clicks bookmark icon in chrome | POST current CFI + paragraph index + text snippet |
| 2 | Toast: "Bookmark saved" | — |
| 3 | User opens BookmarkPanel | List loads via GET |
| 4 | User clicks "Go to" | `viewerRef.navigateTo(cfi)`, panel closes |
| 5 | User clicks trash icon | DELETE bookmark, list refreshes, toast "Bookmark removed" |

### Explainer History Flow

| Step | Action | Result |
|------|--------|--------|
| 1 | User opens ExplainerPanel | Default tab is "Current" |
| 2 | User clicks "History" tab | GET `/api/explainers/history?bookId=xxx` |
| 3 | List shows all explainers for this book | Newest first |
| 4 | User clicks "Go to context" | Navigates to book detail, section, or passage CFI |
| 5 | User clicks an in-flight entry | Nothing (entry is not clickable until complete) |

### Keyboard Shortcuts (Phase 4 additions)

| Key | Action |
|-----|--------|
| `Escape` | Close any open panel (Sheet); if none, hide floating toolbar; if none, navigate back |
| `Cmd/Ctrl+K` | Open SearchPanel |
| `Cmd/Ctrl+F` | Open SearchPanel (alias) |
| `b` | Toggle BookmarkPanel (deferred to v2 — not in v1) |

### Mouse / Touch

| Action | Result |
|--------|--------|
| Click bookmark icon | Save bookmark at current position |
| Click search icon | Open SearchPanel |
| Click highlight in text | Nothing in v1 (v2: open note popover) |
| Long-press highlight | Nothing in v1 (v2: delete option) |

### Accessibility

- Floating toolbar: `role="toolbar"`, `aria-label="Text selection actions"`
- Highlight button: `aria-label="Highlight selected text"`
- Explain button: `aria-label="Explain selected passage"`
- Search input: `aria-label="Search in book"`, `aria-describedby="search-hint"`
- Search results: `role="list"`, each result `role="listitem"`
- Bookmark list: `role="list"`, each bookmark `role="listitem"`
- History list: `role="list"`, each entry `role="listitem"`
- Tabs: `aria-label="Explainer views"` on TabsList
