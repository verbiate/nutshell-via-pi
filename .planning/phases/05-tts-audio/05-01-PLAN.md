Implement:
1. `computeTtsContentHash(text: string): string` — SHA-256 of raw text only (no prompt version).
2. `chunkText(text: string, maxChars: number): string[]` — split by paragraph boundary (`\n\n` or `\n`), keeping each chunk under `maxChars`. If a single paragraph exceeds `maxChars`, split it at the nearest word boundary within the limit. Never truncate mid-word.
3. `getTtsAudio(params: TtsAudioLookup)` — `db.ttsAudio.findUnique({ where: { contentHash_language_voiceId_model: { ... } } })`.
4. `createTtsAudio(data: TtsAudioCreateInput)` — `db.ttsAudio.create({ data })`.
5. `getTtsProviderConfig(provider: "elevenlabs" | "fal", userType: string)` — `db.ttsProviderConfig.findUnique({ where: { provider_userType: { provider, userType } } })`.
6. `generateTtsAudio(params: GenerateTtsParams): Promise<GenerateTtsResult>` — cache-first orchestrator:
   - Fetch the book row: `const book = await db.epubFile.findUnique({ where: { id: bookId } }); if (!book) throw new Error("Book not found");`
   - Call `extractSectionText(book.epubPath, sectionHref)` to get source text.
   - Compute `contentHash = computeTtsContentHash(sourceText)`.
   - Resolve `language` from `book.language`.
   - Call `getTtsProviderConfig(provider, tier)` for both "elevenlabs" and "fal". Pick the first provider that has a non-null `apiKey`. If none found, throw `Error("TTS provider not configured for tier: " + tier)` with statusCode 503.
   - Use the config's `voiceId` and `model`.
   - Check cache via `getTtsAudio({ contentHash, language: book.language, voiceId, model })`.
   - **Cache hit**: return `{ cached: true, audioId: cached.id, url: storage.getUrl(cached.storagePath) }`.
   - **Cache miss**:
     - Chunk text if needed: ElevenLabs limit is 5000 chars for multilingual v2, 10000 for turbo. Use `maxChars = 5000` as safe default.
     - For each chunk, call the appropriate provider function (see Task 3).
     - Concatenate all chunk buffers into a single `Buffer` using `Buffer.concat(buffers)`.
     - Storage path: `tts/${contentHash}/${voiceId}_${model}.mp3`.
     - `storage.write(path, concatenatedBuffer)`.
     - `createTtsAudio({ contentHash, language: book.language, voiceId, model, provider, storagePath: path, fileSize: concatenatedBuffer.length })`.
     - Wrap cache write in `try/catch`. On Prisma unique constraint error (`P2002`), swallow and re-query `getTtsAudio` to return the existing row.
     - Return `{ cached: false, audioId: newRow.id, url: storage.getUrl(path) }`.

**`callElevenLabs(params)`:**
```typescript
export async function callElevenLabs(params: {
  text: string;
  voiceId: string;
  modelId: string;
  apiKey: string;
  signal?: AbortSignal;
}): Promise<Buffer> {
```
- POST to `https://api.elevenlabs.io/v1/text-to-speech/${params.voiceId}/stream`
- Headers: `Accept: audio/mpeg`, `Content-Type: application/json`, `xi-api-key: ${params.apiKey}`
- Body: `{ text: params.text, model_id: params.modelId }`
- Pass `signal` to the `fetch()` call.
- If response not ok, throw `Error("ElevenLabs error " + response.status + ": " + await response.text())`.
- Return `Buffer.from(await response.arrayBuffer())`.

**`callFalAi(params)`:**
```typescript
export async function callFalAi(params: {
  text: string;
  modelId: string;
  voiceId?: string;
  apiKey: string;
  signal?: AbortSignal;
}): Promise<Buffer> {
```
- POST to `https://fal.run/${params.modelId}`
- Headers: `Authorization: Key ${params.apiKey}`, `Content-Type: application/json`
- Body: `{ text: params.text, voice: params.voiceId }` (only include `voice` if `voiceId` is defined)
- Pass `signal` to the `fetch()` call.
- If response not ok, throw `Error("fal.ai error " + response.status + ": " + await response.text())`.
- Parse JSON response. Extract `audio.url` (the response shape is `{ audio: { url: string } }`).
- Fetch the audio URL: `const audioRes = await fetch(audioUrl)`.
- Return `Buffer.from(await audioRes.arrayBuffer())`.
</action>
