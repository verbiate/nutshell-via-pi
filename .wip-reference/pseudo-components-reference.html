A reading surface with a quiet voice

Every control here is drawn in the Nutshell system: warm paper, espresso ink, a coral-to-magenta accent reserved for progress and primary action, and a lavender ring for the active tool. UI text is set in DM Sans; book copy and headlines are set in IBM Plex Serif.
